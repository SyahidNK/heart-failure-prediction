# ❤️ Heart Failure Prediction

Machine Learning project to predict heart disease risk using health indicators.

## Model
Extra Trees Classifier

## Techniques
- Data preprocessing
- SMOTE for imbalance handling
- Ensemble learning

## Deployment
Streamlit Web App

## Features
- Risk prediction
- Risk category (Low / Moderate / High)
- Medical-style gauge chart visualization

pip install -r requirements.txt
